# **App Name**: StudyVerse

## Core Features:

- Live Class Streaming: Allow instructors to host live classes with interactive chat and Q&A.
- Recorded Class Upload: Enable instructors to upload pre-recorded class videos.
- Content Management: Organize notes, lectures, and other resources by course.
- Subscription Management: Control user access to content based on subscription status.
- Free Tier Access: Offer a limited set of lectures and courses accessible without a subscription.
- Personalized Recommendations: AI-powered tool to suggest relevant courses based on user learning history and interests, incorporating similar learning styles and patterns of other users who have watched these lectures.
- Progress Tracking: Track student progress across courses.

## Style Guidelines:

- Primary color: Deep Indigo (#4B0082) for a scholarly and focused feel.
- Background color: Light Lavender (#E6E6FA), offering a gentle, neutral backdrop.
- Accent color: Muted Plum (#8E44AD) to highlight key interactive elements.
- Body and headline font: 'Literata', a serif font for comfortable reading and a literary feel.
- Use clean and professional icons for course navigation and content categories.
- Employ a clear and structured layout to aid in learning and navigation.
- Subtle animations when marking lectures as complete or achieving milestones.
